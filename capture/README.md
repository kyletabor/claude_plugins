# Capture

Zero-friction personal inbox for tasks, bookmarks, ideas, follow-ups, and research items.

## What It Does

Capture lets you quickly save anything to a personal inbox without interrupting your flow. It automatically categorizes items, extracts URLs, infers tags, and cleans up titles. Use it whenever you want to remember something, save a link, note an idea, or track a follow-up without context-switching into a heavier task tracker.

## Components

- **Skills:** `capture` -- quick-capture workflow with automatic categorization
- **MCP Tools:** `capture`, `list`, `search`, `digest`, `update`, `done`, `delete`, `promote`

## Usage

Trigger with `/capture <text>` or natural language like:

- "Remember this"
- "Bookmark this"
- "Save this for later"
- "Follow up on X"
- "Idea: what if we..."
- "Look into Y"

### Categories

Items are auto-categorized based on content:

- **bookmark** -- URLs, "check out", "read this"
- **follow-up** -- "follow up", "circle back", "ping"
- **idea** -- "idea:", "what if", "could we"
- **research** -- "look into", "investigate", "compare"
- **task** -- default for anything actionable

### Other Operations

- "What's in my inbox?" -- list items
- "Search captures for X" -- full-text search
- "Give me my digest" -- summary view
- "Mark X as done" -- close an item
- "Promote X to beads" -- escalate to the issue tracker

## Installation

From the kyle-plugins marketplace -- already included if you have kyle-plugins installed.
